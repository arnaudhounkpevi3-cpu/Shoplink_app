Guide rapide ShopLink
Les étapes essentielles pour configurer, publier, partager et suivre votre boutique.
Compléter ma boutique

Allez dans Mon site, renseignez le nom, slogan, WhatsApp, adresse, numéro d’appel, description et logo, puis cliquez sur Sauvegarder.

  • WhatsApp sert aux commandes.
  • Numéro d’appel sert aux appels classiques.
Ajouter un produit

Ajoutez une photo, un nom, un prix, une catégorie et une description. Ouvrez Options avancées pour disponibilité, stock, prix barré ou variantes.

Voir mon site

Le bouton Voir mon site synchronise vos informations et votre catalogue. Ce que vous voyez en mode propriétaire peut contenir un bouton retour dashboard.

  • Produit publié → visible sur le site.
  • Produit supprimé → retiré du site.
Partager mon lien

Cliquez sur Partager mon lien. Le lien est copié dans le presse-papiers. Collez-le dans WhatsApp, Facebook, Instagram ou SMS.

Recevoir une commande

Sur les sites avec panier, le client ajoute des produits, remplit ses infos, crée la commande, puis envoie le résumé sur votre WhatsApp.

  • La référence est propre à chaque boutique.
  • Le message contient produits, quantités, adresse et total.
Comprendre les statistiques

Vues du site : visiteurs issus d’un lien partagé restés au moins 5 secondes. Commandes reçues : commandes créées depuis vos liens partagés.

  • Produits les plus vus = produits consultés.
  • Top produit = produit le plus commandé.
Besoin d’aide ?

Dans Support, cliquez sur Contacter, choisissez le problème rencontré ou décrivez-le, puis envoyez sur WhatsApp.

Compte et connexion

Dans Compte, vous pouvez modifier votre nom, email et WhatsApp. Si vous changez d’email, utilisez le nouvel email à la prochaine connexion.
